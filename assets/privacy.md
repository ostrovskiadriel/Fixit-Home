# Política de Privacidade - FixIt Home

**Última atualização: 13 de novembro de 2025**

Sua privacidade é importante para nós.

## 1. Coleta de Dados

O FixIt Home coleta o mínimo de dados possível. Coletamos:
* [cite_start]**Dados de Aceite:** A data e versão dos termos que você aceitou[cite: 867].
* **Preferências:** Quais lembretes você configurou (armazenado localmente).

[cite_start]Não coletamos dados pessoais sensíveis[cite: 187, 230, 260].

## 2. Uso de Dados

Usamos esses dados apenas para o funcionamento essencial do aplicativo, como decidir se você precisa ver o onboarding novamente.

## 3. Compartilhamento

Não compartilhamos seus dados com terceiros.
